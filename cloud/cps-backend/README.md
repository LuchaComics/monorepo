# Installation

1. Clone the [this project](git@github.com:LuchaComics/cps-backend.git).

2. Setup `S3 / Object Storage` service.

3. Setup `mailgun`.

4. Setup `Stripe`.

5. Copy and paste the `env_sample` into `.env`.

6. Adjust the variables inside your `.env` file.

7. Run the server.
